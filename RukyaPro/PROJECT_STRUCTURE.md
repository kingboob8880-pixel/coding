## 🚀 Как использовать для экспорта в GitHub

### Вариант 1: Через интерфейс системы
1. Откройте `index.html` в браузере
2. Перейдите в **Настройки** → **GitHub Интеграция**
3. Введите ваш **GitHub Token** (Personal Access Token)
4. Укажите имя репозитория (например, `ruqya-pro-backup`)
5. Нажмите **Экспорт всей базы** — система создаст коммит со всеми файлами

### Вариант 2: Вручную через Git
```bash
cd /workspace/RukyaPro

# Инициализация репозитория
git init
git add .
git commit -m "RUKYA PRO v2.1 — Initial commit"

# Добавление удалённого репозитория
git remote add origin https://github.com/YOUR_USERNAME/ruqya-pro.git

# Пуш в GitHub
git branch -M main
git push -u origin main
```

### Вариант 3: ZIP архив для ручной загрузки
```bash
cd /workspace
zip -r RukyaPro_v2.1.zip RukyaPro/
# Загрузите файл RukyaPro_v2.1.zip на GitHub Releases
```

## 📦 Зависимости (CDN в index.html)

Система использует следующие CDN библиотеки (подгружаются онлайн):
- **html2canvas** — для экспорта в PNG
- **jspdf** — для генерации PDF (опционально)
- **Google Fonts** — шрифты Inter и Amiri
- **Phosphor Icons** — иконки интерфейса

Для полной офлайн-работы скачайте эти библиотеки в папку `assets/lib/`.
