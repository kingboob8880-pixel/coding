# RUKYA PRO — Система для специалистов по рукье

**Ash-Shifa · Абу Мухаммад** — профессиональная PWA-система для ведения базы пациентов, диагностики духовных недугов, автоматической сборки планов лечения и мониторинга состояния.

---

## 📁 Структура проекта

```
RukyaPro/
├── index.html          # Главный файл приложения
├── css/
│   └── style.css       # Стили (10 тем оформления)
├── js/
│   ├── app.js          # Ядро приложения, навигация, UI
│   ├── storage.js      # IndexedDB хранилище
│   ├── data-loader.js  # База знаний (недуги, органы, программы)
│   ├── diagnosis.js    # Диагностический движок (локальный + AI)
│   ├── builder.js      # Генератор формул рукьи
│   └── plan-builder.js # Конструктор планов лечения
└── README.md           # Этот файл
```

## 🚀 Как запустить

### Вариант 1: Прямое открытие в браузере (рекомендуется)

1. Откройте папку `RukyaPro` на вашем компьютере
2. Дважды кликните на файл `index.html`
3. Приложение откроется в браузере

**Преимущества:**
- Работает без интернета
- Не требует сервера
- Все данные хранятся локально в браузере

### Вариант 2: Через локальный сервер

Если хотите использовать все функции (например, AI-диагностику):

```bash
# Python 3
cd RukyaPro
python -m http.server 8080

# Node.js (если установлен)
npx serve .

# PHP
php -S localhost:8080
```

Затем откройте в браузере: `http://localhost:8080`

## 📱 Установка как приложение (PWA)

### На компьютере (Chrome/Edge):
1. Откройте приложение в браузере
2. Нажмите на значок ⋮ (три точки) → «Сохранить и поделиться» → «Создать ярлык...»
3. Поставьте галочку «Открывать в отдельном окне»
4. Нажмите «Создать»

### На телефоне (Android/iOS):
1. Откройте приложение в браузере
2. Нажмите меню → «Добавить на главный экран»
3. Подтвердите установку

## 🎯 Основные возможности

### 1. Новый приём (Мастер из 3 шагов)
- **Шаг 1:** Ввод данных пациента (имя, возраст, пол, телефон, жалобы)
- **Шаг 2:** Диагностика через симптомы или AI (DeepSeek)
- **Шаг 3:** Выбор программы и создание плана лечения

### 2. База пациентов
- Список всех пациентов с карточками
- Поиск и фильтрация по статусу/группе
- Детальная информация о каждом пациенте
- История сеансов и анкет

### 3. Планы лечения
- Автоматическая генерация по диагнозу
- 9 встроенных программ (сихр, джинн, сглаз, васваса и др.)
- Три уровня детализации (Краткий/Средний/Полный)
- Печать планов в оптимизированном формате

### 4. Диагностика
- **Локальная:** Анализ симптомов без интернета
- **AI:** Отправка жалоб в DeepSeek API (требуется ключ)
- Предварительный диагноз в реальном времени
- Оценка тяжести и уверенности

### 5. Библиотека программ
- Просмотр всех встроенных программ
- Информация о длительности и назначении
- Быстрое применение к пациенту

### 6. Настройки
- 10 тем оформления (Classic Gold, Emerald, Night, Royal...)
- Сохранение имени лекаря
- DeepSeek API ключ для AI-диагностики
- Экспорт/импорт всей базы данных

## 🗄️ Хранение данных

Все данные хранятся **локально в браузере** через IndexedDB:

- Пациенты
- Планы лечения
- Заключения
- Группы
- Визиты
- Анкеты
- Настройки

### Резервное копирование

1. Перейдите в **Настройки**
2. Нажмите **«📥 Экспорт базы»**
3. Сохраните JSON-файл в надёжное место

Для восстановления:
1. Нажмите **«📤 Импорт базы»**
2. Выберите ранее сохранённый файл

## 🔌 AI-диагностика (DeepSeek)

Для использования AI-диагностики:

1. Получите API ключ на [platform.deepseek.com](https://platform.deepseek.com)
2. В приложении перейдите в **Настройки**
3. Введите ключ в поле **DeepSeek API**
4. Нажмите **«Сохранить ключ»**

Теперь при создании пациента можно выбрать режим **«AI-анализ жалоб»**.

- Религиозная васваса
- Страх, печаль, гнев

## 🛠️ Технические детали

- **Frontend:** Vanilla JS (без фреймворков)
- **Хранилище:** IndexedDB
- **Стили:** CSS3 с переменными
- **Шрифты:** Inter (интерфейс), Amiri (арабский)
- **PWA:** Service Worker, манифест
- **Офлайн:** Полная работа без интернета
