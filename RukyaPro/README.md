## 🚀 Быстрый старт

### Запуск приложения

1. **Откройте в браузере:**
   ```bash
   # Просто откройте файл index.html в любом современном браузере
   # Или используйте локальный сервер:
   python3 -m http.server 8080
   # Затем перейдите на http://localhost:8080
   ```

2. **Установка как PWA:**
   - Откройте приложение в Chrome/Edge/Safari
   - Нажмите "Установить приложение" в адресной строке
   - Приложение будет доступно офлайн

#### 💎 Премиум темы (5)

### Управление темами:

1. **Переключение тем:**
   - Откройте Настройки → Темы оформления
   - Выберите категорию или прокрутите все темы
   - Кликните на тему для применения

2. **Конструктор палитры:**
   - Настройте основные цвета вручную
   - Автогенерация палитры из основного цвета
   - Сохранение пользовательской палитры

3. **Автопереключение:**
   - Включите в настройках
   - Тема меняется автоматически по времени суток

4. **Высокий контраст:**
   - Режим для слабовидящих
   - Увеличенные границы и контрастность

## 🔧 Основные возможности

### 1. Мастер нового приёма (3 шага)
- Карточка пациента с фото и данными
- Жалобы свободным текстом + голосовой ввод
- Опросник 35 симптомов (8 групп)
- Живой предварительный диагноз
- Импорт JSON от AI
- 9 встроенных шаблонов курсов

### 2. Диагностика
- **Локальный анализ** — без интернета
- **AI-диагностика** — DeepSeek API
- **Ручной ввод** — выбор симптомов
- **Импорт JSON** — из внешних систем

### 3. Планы лечения
- Автоподбор программы по диагнозу
- 3 уровня детализации (Short/Medium/Full)
- Адаптация по тяжести (1-5)
- Лимит формул (слайдер)
- Курс духовных зажимов (опционально)

### 4. База пациентов
- Поиск и фильтрация
- Статусы (активный/пауза/завершён)
- Прогресс лечения (%)
- Экспорт карточки в PNG
- Мягкое удаление и архивирование

### 5. Мониторинг
- 15 быстрых знаков symptoms
- Журнал сеансов (до 100 записей)
- Динамика симптомов
- Анкеты (первичные и повторные)

## 💾 Хранение данных

### IndexedDB (локально в браузере)
- **patients** — пациенты
- **plans** — планы лечения
- **questionnaires** — анкеты
- **settings** — настройки
- **history** — аудит изменений
- **groups** — группы пациентов

### Резервное копирование
- Экспорт всей базы в JSON
- Импорт из JSON
- Локальное шифрование (опционально)
