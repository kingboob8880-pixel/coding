## 📋 Использование Новых Функций

### 1. Смена Темы
1. Откройте **Настройки** (⚙️)
2. Раздел "Тема оформления"
3. Выберите из 30 тем или создайте свою
4. Включите автопереключение по времени

### 2. Экспорт в PNG/PDF
1. Откройте карточку пациента или план
2. Нажмите кнопку "📤 Экспорт"
3. Выберите формат (PNG/HTML/PDF)
4. Файл сохранится с текущей темой

### 3. AI Конвертер Жалоб
1. В мастере нового приёма нажмите "🤖 AI: Текст → JSON"
2. Введите текст жалобы свободной формой
3. Выберите режим (DeepSeek API или локальный)
4. Получите структурированный JSON
5. Примените в мастер приёма

### 4. GitHub Интеграция
1. Настройки → раздел "GitHub"
2. Введите токен, владельца, репозиторий
3. Нажмите "Подключиться"
4. Используйте кнопки:
   - 📤 Экспорт в GitHub
   - 📥 Импорт из GitHub
   - 🔄 Синхронизация

### 5. Валидация UI
Откройте консоль браузера (F12) и выполните:
```javascript
UIValidator.runFullValidation()
```

## ⚙️ Настройка GitHub Token

1. Создайте токен на GitHub:
   - Settings → Developer settings → Personal access tokens
   - Generate new token (classic)
   - Выберите scopes: `repo`, `gist`
   - Скопируйте токен

2. Вставьте токен в настройках RUKYA PRO

## 📊 Технические Характеристики

- **Хранение**: IndexedDB (локально в браузере)
- **PWA**: Работает офлайн, устанавливается как приложение
- **Анимации**: CSS3 + View Transitions API
- **Экспорт**: html2canvas для PNG, браузерная печать для PDF
- **API**: Fetch API для GitHub и DeepSeek
- **Безопасность**: Данные только локально, токены в localStorage
